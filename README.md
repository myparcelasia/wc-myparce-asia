# MyParcel Asia WooCommerce Plugin

## General Description
The MyParcel Asia WooCommerce plugin seamlessly integrates your WooCommerce store with MyParcel Asia's delivery infrastructure. It automates shipping rate calculations and allows you to generate Air Waybills (AWBs) individually or in bulk, helping you streamline your fulfillment operations.

## Features

- **Customer Checkout Price**: Accurately calculate live shipping rates at checkout based on destination and package weight, with options for flat rate fallbacks and custom markups.
- **To Process and Batch Create AWB**: Intelligently group unfulfilled orders into batches and generate multiple AWBs simultaneously with a single click.
- **Order Page Create AWB**: Manually generate individual AWBs directly from the WooCommerce order details page, allowing you to select specific couriers for special requests.
- **Lane Management**: Periodically sync your shipping lanes to ensure your pricing data remains accurate and up to date.
